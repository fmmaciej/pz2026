## Komendy
`cp` - kopiowanie plików i katalogów

- Kopiowanie katalogu razem z zawartością:
```bash
cp -R katalog1 katalog2
```

- Kopiowanie pliku z zachowaniem uprawnień
```bash
cp -p plik1 plik2
```

- Kopiowanie katalogu z zachowaniem uprawnień
```bash
cp -Rp katalog1 katalog2
```

`mv` - przenoszenie lub zmiana nazwy plików
```bash
mv plik1 plik2
```

`pwd` - pokazuje pełną ścieżkę katalogu
```bash
pwd
```

`trap` -  pozwala wykonać określony kod w odpowiedzi na sygnał wysłany do programu
```bash
trap 'komenda do wykonania' SYGNAŁ
```

`-f` - wymuszenie, wykonuje operację bez pytania o potwierdzenie

`-L plik.txt` - sprawdza, czy plik istnieje i jest linkiem symbolicznym

`-e plik.txt` - sprawdza, czy plik istnieje
