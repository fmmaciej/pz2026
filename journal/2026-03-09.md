# 2026-03-09 - Todo-cli - Program główny

## Cel

- Pair programming funkcji programu `todo-cli`
- Zapoznanie z funkcjami w bibliotekach: `json`, `pathlib`

## Komendy

Pozwala na przekazanie szczegółowych informacji o błędzie:

```python
  raise TodoError(f"Couldn't find the index of item of id: {id}")
```

Znajduje maksymalny ID iterując po elementach tablicy:

```python
  max(item.id for item in self.items) + 1
```

Tworzenie nowej instancji klasy przez parametr `cls`:

```python
  return cls(items=[])
```

Definicja głównej funkcji z opcjonalnym argumentem - pozwala na testowanie z domyślnymi argumentami lub przekazywaniem własnej listy:

```python
  def main(argv: list[str] | None = None):
    if argv is None:
        argv = sys.argv[1:]
```

Konwersja danych aby były spójne z fomatem `json`:

```python
   json.dumps(data, ensure_ascii=False, indent=2)
```

Tworzenie katalogu i wszystkich katalogów nadrzędnych, nie wyrzucając błędu nawet jesli istnieją:

```python
  self.file_path.parent.mkdir(parents=True, exist_ok=True)
```

Definicja listy konwertująca elementy na słowniki poprzez `__dict__`:

```python
  data = [x.__dict__ for x in todos.items]
```
