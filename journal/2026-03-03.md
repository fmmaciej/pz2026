# 2026-03-03 - UV i projekty w python'ie

## Cel

- Poznać podstawowe użycie programu `uv`
- Poznać sposoby instalacji programów: `pytest`, `mypy`, `pre-commit`, `ruff`
- Tworzenie konfiguracji projektu
- Zapoznanie z wirtualnym środowiskiem (`.venv`)

## Komendy

`uv init` - inicjalizowanie nowego projektu UV

```bash
uv init
```

`uv python pin` - ustawienie wersji Python'a

```bash
uv python pin 3.12
```

`uv python install` - instalacja określonej wersji Python'a

```bash
uv python install 3.12
```

`uv python list` - wylistowanie zainstalowanych wersji Python'a

```bash
uv python list
```

`uv add --dev` - dodanie pakietów dla deweloperów

```bash
uv add --dev ruff pytest pre-commit mypy
```

`uv sync` - synchronizacja wirtualnego środowiska z plikiem `pyproject.toml`

```bash
uv sync
```

`uv run` - uruchomienie komendy w wirtualnym środowisku

```bash
uv run python -V
uv run ruff check .
uv run mypy src
uv run pre-commit install
```

`deactivate` - deaktywacja wirtualnego środowiska

```bash
deactivate
```

`cp` - kopiowanie plików i katalogów

```bash
cp <ścieżka do pierwszego pliku kopiowanego> <ścieżka do drugiego pliku kopiowanego> <ścieżka katalogu docelowego>

cp -r <ścieżka do katalogu> <ścieżka docelowa>
```

`mv` - przenoszenie i zmiana nazwy plików/katalogów

```bash
mv pythonProject starter_project

mv tests/{,test}_core.py
```

`^<before>^<after>` - wykonuje ostatnią komendę z wymienionym `before` na `after`

```bash
git config user.name "Imie"
^name "Imie"^email "Email"

# Powyższe linijki złożą się na:

git config user.email "Email"
```

## Pliki konfiguracyjne

- `pyproject.toml` - główny plik konfiguracyjny projektu Python'a
- `.pre-commit-config.yaml` - konfiguracja pre-commit
- `.editorconfig` - konfiguracja edytora
- `.vscode/settings.json` - ustawienia VSCode
