## Komendy

```bash
chmod +x plik.sh -> dodaje uprawnienia
chmod -x plik.sh -> usuwa uprawnienia

r(read) = 4, w(write) = 2, x(execute) = 1
rwx: 4 + 2 + 1 = 7

chmod 763 plik.sh
7 = 4 + 2 + 1 -> rwx (właściciel)
6 = 4 + 2 -> rw (grupa)
3 -> 2 + 1 -> wx (inni)


env -> wyświetla zmienne środowiskowe

export ZMIENNA="blabla" (wielkie litery)
echo $ZMIENNA


history -> lista ostatnich poleceń
history | grep git (wyświetli wszystkie polecenie związane z git)

!! -> powtórzenie ostatniego polecenia
!-2 -> powtórzenie drugiego polecenia od końca
!158 -> odwołanie się do polecenia o numerze 158 w historii

tr ':' '\n' -> zmiana : na \n
```
